## Requirements

- GameMode LaLife : https://github.com/ldlac/La_LifeRP/

## Installation

- Download the files
- Drag the folder in your resources
- add "- admin_menu" in your citmp-server
- Launch your server
- Press F1 for open the Menu (you need to be admin/superadmin)
